# 教訓・修正パターンの記録

## L-001｜キャラ間の上下関係（学年・敬語）を意識する

**修正例**：まーちゃん（2年）→ふれんち（3年）の呼称・口調が「ふれんち君／タメ口」になっていた。

**正しい扱い**：
- まーちゃんは後輩なので、本来は **「ふれんちさん」＋丁寧語（〜ですか／〜ます）** が自然
- ほんわか系の性格でも、敬語は崩さないのが普通

**チェックリスト（脚本・追加台詞時）**：
- [ ] 話者と相手の学年差を確認したか？
- [ ] 後輩→先輩は「さん」付け＋丁寧語
- [ ] 同期同士はあだ名＋タメ口で OK
- [ ] 先輩→後輩は呼び捨て or 「ちゃん／君」＋タメ口
- [ ] 既に同期で親しい間柄が確立してる例外（例：ミオ↔ふれんち、薫↔ふれんち）はそのまま

**学年マップ（参考）**：
- 4年：ミヤコさん、加藤さん（25歳・社会人）
- 3年：ふれんち、ミオ、薫、ユタカ、平野、姫、シェリー、長友、東山、たえやん、愛ちゃん、中野さん、ヨコノ
- 2年：渡部、まーちゃん
- 1年：ヒマリちゃん

## L-002｜キャラの外見・髪型・服装の変化を描写するときは、立ち絵と矛盾させない

**修正例**：3-01 で「ヒマリちゃんは何故かツインテールをやめてポニーテールになっていた」と描写していたが、実際の立ち絵はずっとポニーテール。プレイヤーが「あれ、ツインテールだったっけ？」と混乱する。

**正しい扱い**：
- 立ち絵に固定された外見の特徴（髪型・服装の決定的な要素）は変化させない
- 季節感・時間経過は **「雰囲気がほんの少しだけ大人っぽく」「日焼けしている」「ちょっと痩せた」** など、立ち絵で表現されない／表現されても誤差レベルの曖昧な描写にする
- 服装変更を匂わせるなら、ステージ衣装・特別衣装は CG 化前提

**チェックリスト（脚本・追加台詞時）**：
- [ ] そのキャラの立ち絵の特徴（髪型、服装、メガネの有無、肌の色）と矛盾していないか？
- [ ] 「変化した」と書く場合、立ち絵で表現できない種類の変化か？
- [ ] 確実な変化として描写するなら、CG として用意できるか？
