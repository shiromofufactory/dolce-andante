# BGM リスト・取得ガイド

## ファイル命名ルール

`assets/bgm/{key}.mp3` （拡張子は `.mp3` `.ogg` `.m4a` `.wav` のいずれもOK、自動検出）

## 必要BGM一覧

ゲームエンジンは**背景に応じて自動でBGMを切り替え**ます。下記の `key` のファイルを `assets/bgm/` に置いてください。

| ファイル名 | 用途・雰囲気 | 使われる場面 |
|---|---|---|
| **daily.mp3** | 日常／柔らかいピアノ／ノスタルジック／少しテンポあり | 部室昼夕、ドルチェ、銀杏並木（緑）、廊下 |
| **night.mp3** | 夜・静寂／ピアノソロ／落ち着き | 部室夜、自宅、夜の住宅街 |
| **waiwai.mp3** | 居酒屋／賑やか／ジャズ寄り or アコースティック | 居酒屋「わいわい」 |
| **autumn.mp3** | 秋・少し切ない／弦楽器寄り／中速 | 銀杏並木（黄・散）、学園祭準備 |
| **stars.mp3** | 星空・幻想的／ピアノ＋アンビエント | 長野の河原（合宿の夜） |
| **summer.mp3** | 夏・少し明るく郷愁／ピアノ＋弦 | 合宿、京都、松本 |
| **warm.mp3** | 教会の昼／温かいアコースティック | 教会の中庭（夏祭り） |
| **christmas.mp3** | 静かで荘厳／鈴音／ピアノ | 教会ミサ、イルミネーション、クリスマスシーン |
| **festival.mp3** | 学園祭／少し賑やか・楽しげ | 学園祭「あんだんて」 |
| **emotional.mp3** | 切ない／ピアノソロ／ゆっくり | 病院、告白、別れ |
| **concert.mp3** | 演奏会本番／オーケストラ／パブリックドメイン録音可 | 夏／冬の定期演奏会 |
| **title.mp3** | タイトル画面用／メインテーマ的／ピアノ＋弦 | （オプション）タイトル画面で再生したい場合 |

**最低限の構成**：`daily / night / emotional / concert / christmas` の5曲があれば、ゲーム全編それなりに音楽がついた状態になります（他は任意）。

## おすすめ取得サイト

### ★★★★★ 甘茶の音楽工房（最優先）
https://amachamusic.chagasi.com/

**雰囲気**：ピアノ中心、優しい・切ない・ノスタルジック。本作の水彩風画風と非常に相性が良い。
**ライセンス**：商用OK、クレジット表記が望ましい。
**おすすめジャンル**：「ピアノ」「ヒーリング」「切ない」「やさしい」カテゴリ

候補曲（曲名で検索）：
- `daily` → 「カフェテラスの午後」「窓辺の風」「ぬくもりの記憶」
- `night` → 「夜想曲」「眠りの森」「月光のソナタ」
- `emotional` → 「サクラ咲ク頃」「淡い夢の記憶」「ノスタルジア」
- `christmas` → 「聖夜」「やすらぎのクリスマス」

### ★★★★★ DOVA-SYNDROME
https://dova-s.jp/

**雰囲気**：日本最大のフリーBGM、ジャンル豊富。タグ・気分検索可。
**ライセンス**：商用OK（個別曲のライセンス確認推奨）。

候補：
- 「ピアノ」「切ない」「ノスタルジック」タグで検索
- `summer` → 「夏の終わり」系の曲
- `stars` → 「星空」「夜空」系
- `festival` → 「祭り」「ワクワク」系

### ★★★★ MusMus
https://musmus.main.jp/

**雰囲気**：個人ゲーム制作で定番、商用OK、クレジット表記。
**おすすめ**：「日常」「街」「サブ」シリーズ → daily/waiwai 用に使える曲多数

### ★★★★ BGMer
https://bgmer.net/

**雰囲気**：気分（怖い・楽しい・切ない）から検索可。
**おすすめ**：「切ない」「優しい」カテゴリから emotional / autumn 用

### ★★★ Yung.jp（Blue Sky Label）／ CMSL
http://www.yung.jp/ 
http://classical-sound.seesaa.net/

**雰囲気**：クラシック音楽のパブリックドメイン録音。
**用途**：`concert.mp3` 用に、チャイコフスキー交響曲第5番、サン＝サーンス《ピアノ協奏曲第2番》などの実録音を入手可能。
**注意**：**著作隣接権**（演奏者・録音者の権利）が残っている録音もあるため、配布元の使用許諾を必ず確認してください。

### ★★ HURT RECORD
https://www.hurtrecord.com/

**雰囲気**：ピアノ・ギター・弦楽器寄り、商用OK。

## ファイル形式・サイズについて

- 推奨形式：**MP3**（128〜192kbps 程度で十分。容量と音質のバランス）
- 推奨長さ：**1〜3分**（自動でループ再生されます）
- 推奨ファイルサイズ：1曲あたり **2〜4MB** 程度

WAV/FLAC など重いファイルでも動作しますが、Web 公開時のロード時間が長くなります。

## 配置例

```
assets/bgm/
├── daily.mp3
├── night.mp3
├── waiwai.mp3
├── autumn.mp3
├── stars.mp3
├── summer.mp3
├── warm.mp3
├── christmas.mp3
├── festival.mp3
├── emotional.mp3
└── concert.mp3
```

配置するだけで、自動でゲーム内のシーンに合わせて鳴ります。
クロスフェードは1.2秒で自動切替。

---

## クレジット表記について

サイトの利用規約により、ゲーム内のクレジット画面に下記のような表記が必要になる場合があります（特に MusMus, 甘茶の音楽工房など）：

```
BGM:
  甘茶の音楽工房 (https://amachamusic.chagasi.com/)
  MusMus (https://musmus.main.jp/)
  ...
```

ゲームのクレジット画面（タイトル画面の「クレジット」ボタン）に表示するように後で実装します。利用された曲が確定したら教えてください。

---

## 実装済みの機能

- ✅ BGM の自動ロード（`assets/bgm/{key}.mp3` を検索）
- ✅ 背景に応じた自動BGM切替
- ✅ クロスフェード（1.2秒）
- ✅ ループ再生
- ✅ 同じ曲が連続する場合は再ロードしない
- ✅ ファイルが無ければ無音（ゲームは止まらない）

ファイルを置けばすぐに動きます。何曲か試しに入れてみて、雰囲気を確認してください。
