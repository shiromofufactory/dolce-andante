# 第2章 「空白のひと月」（7月末〜8月末）

## 概要
- 時期：7月末（演奏会翌日）〜8月末
- 役割：各ヒロインルートの「種」を育てる。プレイヤーの夏の過ごし方選択により、シーンのセットが大きく変わる。
- 分岐：最大。Path A〜Dの4系統で展開。
- トーン：少し切なく、少し解放感がある、大学生の夏。

---

## シーン 2-00｜分岐ハブ（演奏会翌日・ふれんち自室）

**[モノローグ]**
　演奏会の余韻も、朝には消える。
　部活はオフ。大学も夏休み。
　――俺は、今年の夏をどう過ごすか、決めなきゃいけない。

**[選択肢 C2-HUB]**（※ 第1章の選択により、表示される選択肢が変わる）
> Path A：免許合宿に行く（第1章でミオに「行く」と返信した場合、ミオ同行が確定）
> Path B：バイト中心で街に残る（加藤さん夏祭り参加がONなら、このPath）
> Path C：旅行に行く（愛ちゃんや姫ちゃん絡み）
> Path D：帰省する（比較的地味、中野さんルート適性）

※ フラグ条件を満たしていないPathは出現しない。
※ どのPathを選んでも第3章に合流するが、好感度の動き方が大きく異なる。
